# 플레이스 순위·지수 분석툴

키워드와 플레이스 ID를 입력하면 **순위 / N지수(n1·n2·n3) / 경쟁강도(C)** 를 분석하는 Next.js 앱입니다.

## 기능

- **내 플레이스 순위**: 키워드 + 플레이스 ID 입력 시 몇 위인지 표시
  - 30위 안에 없으면 `page=1 → 2 → 3` 순차 조회로 **최대 90위까지** 탐색
- **내 플레이스 N지수**: n1, n2, n3 값 표시
- **경쟁강도(C)**: 기준값 `0.1868` 대비 판정
  - C < 0.1868 → 🟢 황금 키워드 (진입 유리)
  - C ≥ 0.1868 → 🔴 경쟁 치열 (높을수록 진입 어려움)
- **N지수 그래프**: 상위 1~2페이지(최대 60위) 모든 플레이스의 n1·n2·n3 막대 그래프 (내 플레이스 위치는 빨간 점선으로 표시)
- **상위 10개 대표키워드**: 1~10위 플레이스의 대표키워드 목록

## 기술 스택

- Next.js 15 (App Router) + TypeScript
- Tailwind CSS 4
- Recharts (막대 그래프)
- 서버 API 라우트(`/api/place`)로 외부 API 프록시 (CORS 회피 + 순차 페이지 조회)

## 데이터 출처

`https://place.diccmain.workers.dev/api?q={키워드}&page={페이지}` (페이지당 30개)

## 실행

```bash
npm install
npm run dev      # 개발 서버 (기본 http://localhost:3000)
npm run build    # 프로덕션 빌드
npm start        # 프로덕션 실행
```

## 사용법

1. **키워드** 입력 (필수) — 예: `강남맛집`
2. **플레이스 ID** 입력 (선택) — 예: `1046192699` (순위/내 지수 분석용)
3. **분석** 클릭
